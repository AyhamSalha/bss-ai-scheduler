## Demo-Notizen (Sprint 1)

**Funktionalitäten:**
- Der Chat akzeptiert Texteingaben.
- Die Nachricht wird per POST an den FastAPI-Server geschickt.
- Die Antwort ist eine Dummy-Nachricht („Danke für deine Nachricht!“).
- Alle Eingaben werden in `chat.db` gespeichert (mit Benutzer, Nachricht, Timestamp).

**Frontend-Details:**
- HTML + CSS + JavaScript
- Button zur Anzeige/Ausblendung der Chatbox
- Layout im dunklen Stil

**Backend:**
- FastAPI, SQLite, Pydantic
- Zwei Endpunkte: `/` (Status), `/eintrag` (POST)
